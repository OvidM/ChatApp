# ChatApp

BlazorServer Chat Application Developed using <b>SignalR</b> for the Real Time Functionality, <b>ASP.NET Core Identity</b> for User <b>Authentication</b> and <b>Authorization</b>. I use 2 SQLite Databases: <b>Auth.db</b>, with which I communicate using <b>Entity Framework Core</b>, and <b>Messages.db</b> with which I communicate using <b>SQL Queries</b>
